# Welcome to Aelin

欢迎你使用 Aelin。

如果用一句更容易记住的话来概括它：Aelin 不是“只回答这一次问题”的聊天窗口，而是一个会随着你使用而逐步理解你、记住你、辅助你长期决策的助手系统。

## Aelin 解决什么问题

在很多真实场景里，真正的难点并不是“找一条信息”，而是“持续跟踪一类信息并在合适时机做出判断”。

你可能会遇到这些情况：

- 关注对象分散在多个平台，切换成本高。
- 普通聊天工具每次都从头开始，历史价值难以沉淀。
- 你知道自己关心什么，但很难稳定地追踪变化。

Aelin 的工作方式，正是围绕这些长期问题设计的。

## Aelin 与普通 Chat 的区别

- 普通 Chat 更擅长一次性问答：提问、回答、结束。
- Aelin 更强调连续性：提问、证据、跟踪、记忆、再次追问。

这意味着你第二次、第三次回来时，不必重复介绍背景。系统会在可控范围内延续上下文，让对话更像“同一件事的持续推进”，而不是“重复开新局”。

## 你可以期待的核心能力

Aelin 当前的能力可概括为五个模块：

- `Agent Chat`：更自然的流式回答、证据引用和可持续会话。
- `Web + Local Memory`：同时理解你的历史信息和外部最新信息。
- `Long-term Tracking`：把一次关注，变成持续更新的任务。
- `Proactive Interaction`：在关键变化出现时主动提醒，而非被动等待。
- `Device Interaction`：在合规权限下感知并调整设备状态。

## 三个关键词

- `Answer-first`：先给可执行结论，再展开依据与建议。
- `Long-term memory`：把高价值信息沉淀为后续对话可复用的上下文。
- `Tracking`：由你确认后，系统才建立持续跟踪，始终保持可控。
