# FAQ

## Q1: Aelin 为什么有时会联网，有时不会？

A: 默认是模型自动决策。问题涉及最新事实时，联网概率更高；问题偏历史或个人上下文时，更可能优先本地记忆检索。你可以把它理解为“根据问题性质动态选路”。

## Q2: 跟踪会自动开启吗？

A: 不会。Aelin 只会提出建议，是否建立长期跟踪由你确认。这个设计是为了保证可控性，避免系统擅自扩张任务范围。

## Q3: 为什么有时回答不够深入？

A: 常见原因包括：模型配置异常、外部证据不足、问题描述过宽。建议先检查 LLM 连通性，再把问题具体化（对象、时间、范围）。

## Q4: 可以自定义 LLM 提供商吗？

A: 可以。Aelin 支持自定义 Provider / Base URL / Model。建议先完成连通性测试，再进行参数微调。

## Q5: 记忆会不会无限增长？

A: 当前通过分层与摘要机制控制规模。后续版本会继续优化压缩、淘汰与质量评估策略。

## Q6: 主动提醒太多怎么办？

A: 优先收敛跟踪范围，关闭低价值主题，保留高优先级对象。提醒质量的关键不在“数量”，而在“相关性”。

## Q7: 我需要每次都复核结果吗？

A: 对一般信息场景可以直接使用；对高风险决策（业务、财务、合规）建议保留人工复核，这是一种稳妥而专业的使用方式。
