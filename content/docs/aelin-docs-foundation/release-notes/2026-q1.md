# Release Notes 2026 Q1

## 版本定位

本次不是功能发布，而是文档基线重建：

- 以 `MercuryDesk` 当前源码为唯一事实来源
- 将旧版通用文案替换为可落地、可核验的 Aelin 文档
- 明确已实现能力、技术特点与当前边界

## 本次文档更新范围

1. 信息架构
- 重写 `README` 与 `NAVIGATION`
- 统一站点阅读路径与能力映射

2. 功能文档
- Chat SSE、引用、动作建议、链路追踪
- 多源接入与异步同步
- 跟踪系统与自治调度
- Desk 观察台与内容打标
- 媒体摄取与质量门禁
- 文件记忆与日记树
- 设备控制、专注模式、Electron 壳与桌宠引擎

3. 指南文档
- LLM 配置（规则 + OpenAI-compatible）
- 追踪闭环操作指南
- Web/桌面/移动访问与打包说明

4. 参考文档
- API 按域总览
- 存储与记忆结构
- FAQ
- 已知限制与规避建议

## 明确写入的边界

- 知乎入口仅前端可见，后端无独立连接器。
- 设备模式只保证 `meeting/focus/sleep/normal/default`，其它值降级处理。
- 鉴权存在本地单用户 fallback，偏本地部署场景。

## 下个周期建议

1. 在文档中加入接口字段示例（request/response）
2. 为媒体摄取补充更细的错误码对照表
3. 若后端补齐知乎连接器，更新接入矩阵与 Known Issues
